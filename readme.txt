🛠 How to Use (Basic Guide)
--------------------------

1️⃣ Open 'index.html' in any browser to preview locally.

2️⃣ To host the website for FREE:

   a. Create a GitHub account at https://github.com  
   b. Create a new repository (name it e.g. `akhilnews`)  
   c. Upload all files (index.html, styles.css, logo.png)  
   d. Go to repository settings → Pages → Set Source to `main` branch & `/root`  
   e. Your site will go live at: https://yourusername.github.io/akhilnews/

3️⃣ To Add New Article:
   - Open `index.html`
   - Duplicate the <article> block inside <section class="articles">
   - Change heading & paragraph text
   - Save and upload again on GitHub
